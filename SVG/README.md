# Barefoot Labs Logo Files

This folder contains 20 SVG logo files across 5 different design concepts.

## File Structure

Each concept has 4 variations:
- `concept-X-mark-dark.svg` - Icon only, for dark backgrounds
- `concept-X-mark-light.svg` - Icon only, for light backgrounds
- `concept-X-horizontal-dark.svg` - Icon + wordmark, for dark backgrounds
- `concept-X-horizontal-light.svg` - Icon + wordmark, for light backgrounds

## Concepts

### Concept 1: Overlapping Circles
Two overlapping circles suggesting heel and toe, with the accent color on the smaller circle.

### Concept 2: Asymmetric Oval
A single rotated oval with a subtle inner arch cutout, representing an abstracted footprint.

### Concept 3: Arc Step
Curved side profile of a step with a small toe circle above.

### Concept 4: Step Progression
Dots arranged in a walking/stepping pattern with a main footprint and toe details.

### Concept 5: Angular Geo
Modernist geometric shape with angular edges and an inner arch cutout.

## Colors

- **Navy**: #0f172a (primary color for light backgrounds)
- **Sky Blue**: #38bdf8 (accent color)
- **White**: #ffffff (primary color for dark backgrounds)

## Usage Recommendations

- **Favicon**: Use the `-mark-` versions at 16px, 32px, or 180px
- **App Icons**: Use the `-mark-` versions, scalable
- **Website Header**: Use the `-horizontal-` versions
- **Social Media**: Use either `-mark-` or `-horizontal-` depending on space
- **Print**: All files are vector SVG and can scale infinitely
